What's new?

The current version of this app releases the following features:

  • Receiving in new inventory
  • Printing plant labels
  • Support for Socket Mobile barcode scanners
  • Support for Brother QL Bluetooth printers

What's next?

Future versions of the app will release the following features:

  • Changing inventory (which is different from receiving) - please use MandyMae in the meantime
  • Assigning bins to products
  • Improvements to the search function in the catalog
  • Release of a Windows version, supporting shipping functions
